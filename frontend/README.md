# Docker tutorial

This template is an extremely barebones Spring Boot/Vue 3 project that exists only so users can try out some Docker stuff

# Running the application

Load this project up in IntelliJ, and it should be able to run the Spring Boot side of things through the Run button in the top right

Then open a terminal window, navigate to /frontend, and run npm install; then you can run npm run dev and it'll tell you where the application has been deployed
